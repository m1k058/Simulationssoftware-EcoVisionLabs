 Dieses Dokument beschreibt die **features** der Software für  **IPJ1**.  

## 1. Features

1. **Einlesen von `.csv`-Datein von SMARD** (© Bundesnetzagentur 2025) **Stündlich und Viertelstündlich** (erforder auskomentierung von Code Teilen)
2. **Generierung von Balkendiagrammen mit eingelesenen Daten**
    - Filterung von verschiedenen Energiequellen
    - Gruppierte Energiequellen (Konventionelle, Erneuerbare, Low_Carbon, Speicher)
    - Farbliche anpassung der einzelnen Energiequellen
    - Zeichtlicher Raum auswählbar
3. Speicherung als `.png` mit Zeitstempel
4. **KEIN UI**
